You're going to write a random_enemy() function that creates an enemy with random properties. This will be super useful later for adding variety to your game!

ğŸ“‹ Task: Create the random_enemy() Function

Add this function to your code (you can put it right after your create_enemy() function):

function random_enemy()
  local enemy = create_enemy("enemy_ship", 32, 32)

  -- Random position anywhere on screen
  enemy.x = (math.random() - 0.5) * 200
  enemy.y = (math.random() - 0.5) * 200

  -- Random velocity
  enemy.vx = (math.random() - 0.5) * 2
  enemy.vy = (math.random() - 0.5) * 2

  return enemy
end

ğŸ” How This Works

  • math.random() - Returns a random number between 0 and 1
  • (math.random() - 0.5) - Shifts range to -0.5 to 0.5 (both positive and negative)
  • * 200 - Scales position to cover the screen (-100 to 100)
  • * 2 - Scales velocity to -1 to 1 for movement speed
  • return enemy - Returns the newly created random enemy
