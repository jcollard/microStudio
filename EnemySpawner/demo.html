‚ú® Features

  • Syntax Highlighting: Lua code is automatically highlighted using Highlight.js in both code blocks AND diffs
  • Copy Buttons: Click "Copy Code" to copy to clipboard with visual feedback
  • Side-by-Side Diff: Clearly shows what was removed (red) and added (green) with syntax highlighting
  • Line Numbers: Both sides of the diff show line numbers for easy reference
  • Clean Design: Simple, focused styling that doesn't distract from the code

üìù Usage

To use these functions in your lesson pages:

  1. Add the CDN links to your HTML <head>
  2. Include codeblocks.js and codeblocks.css
  3. Create a <div> with a unique ID
  4. Call renderCodeBlock() or renderDiffBlock() in a <script> tag
