Understanding Table Indexing in Lua

The expression SIZES[math.random(1, 3)] breaks down like this:

  • SIZES = {24, 32, 40} - Creates a table with three size options using curly braces { }
  • math.random(1, 3) - Generates 1, 2, or 3 randomly
  • SIZES[1] - Returns 24 (first element)
  • SIZES[2] - Returns 32 (second element)
  • SIZES[3] - Returns 40 (third element)

Important: Lua uses 1-based indexing, so the first element is at index 1, not 0!
